# TrustAgent

A simple AI tool that checks whether a claim matches widely known public facts.

Runs locally with Ollama.


**How It Works**
- Enter a claim
- Facts are retrieved
- Claim is marked:
  - Consistent
  - Inconsistent
  - Uncertain



**Scoring**
- Consistent → 0.9
- Inconsistent → 0.1
- Uncertain → 0.5
